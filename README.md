# saas-product-builder
An end-to-end saas product building framework for solo builders

This app is intended to be executed within the terminal


## Install

```bash
$ npm install --global saas-product-builder
```

## CLI

```
$ saas-product-builder --help

  Usage
    $ saas-product-builder
  Options
    --name  Your name

  Examples
    $ app --name=Jane
    Hello, Jane
```
